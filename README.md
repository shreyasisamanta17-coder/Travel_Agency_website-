# Travel Agency Management System

A full-stack **Travel Agency Website** developed using **HTML, CSS, Bootstrap, PHP, and MySQL**. The platform allows users to explore travel packages, manage bookings, and save favorite destinations, while providing administrators with tools to monitor and manage customer bookings.

## Features

### User Module

* User Registration and Login Authentication
* Browse Available Travel Packages
* View Package Details
* Book Travel Packages
* Add Packages to Wishlist
* Manage Personal Bookings
* Responsive User Interface

### Admin Module

* Secure Admin Login
* View and Manage Booking Records
* Monitor Package Reservations
* Track Customer Booking Activity
* Manage Travel Package Information

## Technologies Used

* **Frontend:** HTML5, CSS3, Bootstrap
* **Backend:** PHP
* **Database:** MySQL
* **Development Environment:** Antigravity

## Project Highlights

* Role-based access for Users and Administrators
* Dynamic package booking system
* Wishlist functionality for users
* Database-driven application
* Responsive design for different screen sizes
* Secure login and session management

## Purpose

This project was developed to demonstrate full-stack web development concepts, including user authentication, database integration, CRUD operations, session handling, and responsive web design in a real-world travel booking application.
